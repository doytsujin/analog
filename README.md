analog
------

`analog` could be an analogical (shell) scripting language.

```bash
b = 99

|?| b >> 0
    :: b "bottles of ink on the wall."
       b "bottles of ink.
       Take one down
            scribble around..."
    -- b
    :: b "bottles of ink on the wall."
    % (sudo reboot)
```

The [reference](./docs/reference.md) provides a simplified overview.
